Detailed Description

Linked-list based queue for request_t elements.

This module implements FIFO queue where each node stores a pointer to a request_t. It supports enqueue and dequeue operations.

Typical usage:

queue_t* q = create_queue();
enqueue(q, req1);
enqueue(q, req2);
request_t* r = dequeue(q);
// ...
