Detailed Description

Minimal linked-list queue for request_t elements.

This module defines the types and functions for a simple FIFO queue used to store request_t objects. Each node holds a pointer to a request_t, which contains information about a client request.

The queue supports:

  • Creating a new queue with create_queue
  • Adding a request with enqueue
  • Removing the oldest request with dequeue
